# Tennis Matching
テニスプレイヤーへ向けたマッチングアプリです。<br>
参加者を募る投稿、それに対して参加のリクエストを送り、許可された場合のみトークルームに参加という形式でマッチングを成立させています。<br>
サイトurl : http://shrouded-waters-22362.herokuapp.com/
